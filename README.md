# Contract89 QR Redirect

Статическая QR-страница для Netlify.

## Как использовать

1. Создай новый репозиторий на GitHub.
2. Загрузи в корень репозитория файлы:
   - `index.html`
   - `netlify.toml`
3. В Netlify выбери:
   - Add new site
   - Import an existing project
   - GitHub
   - нужный репозиторий
4. Build command оставь пустым.
5. Publish directory: `.`
6. Deploy site.

После деплоя Netlify даст ссылку вида:

`https://site-name.netlify.app`

Именно её нужно зашить в QR-код.
